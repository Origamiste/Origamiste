L'Origamiste

Introduction

Ce site web a comme but de vous donner les bases requises pour vous aventurer dans l'art qu'est l'origami. Ce site web vous informera sur les bases, sur les papiers, et sur la procédure de création. Il est alors important que vous sachiez que la plupart des termes vont être en anglais puisque le contenu disponible sur l'origami utilise davantage cette langue que le français.


Qui suis-je?

Je suis Yosuf Azami, un élève qui à retrouver sa passion de son enfance pour l'art de l'origami.
